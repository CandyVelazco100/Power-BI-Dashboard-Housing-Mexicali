# Power BI Dashboard - Housing Mexicali

## 📊 Descripción

Dashboard interactivo desarrollado en Power BI para analizar el mercado de vivienda en Mexicali, Baja California.

El proyecto permite explorar indicadores relevantes del sector inmobiliario mediante visualizaciones dinámicas que facilitan la toma de decisiones basada en datos.

## 🎯 Objetivos

- Analizar tendencias del mercado inmobiliario en Mexicali.
- Identificar zonas con mayor oferta de vivienda.
- Comparar precios y características de propiedades.
- Facilitar la exploración interactiva de datos.

---

## ✨ Características

- Visualizaciones interactivas.
- Segmentación por ubicación.
- Análisis de precios.
- KPIs principales del mercado.
- Filtros dinámicos.

---

## 🛠 Tecnologías Utilizadas

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel / CSV

---

## 📂 Fuente de Datos

Los datos utilizados corresponden a información del mercado inmobiliario de Mexicali.
